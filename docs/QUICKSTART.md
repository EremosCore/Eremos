# Quickstart Guide

Follow these steps to set up Eremos Core and launch your first swarm agent.

---

## 1. Prerequisites

- Node.js >= 18
- npm >= 9
- Git

---

## 2. Clone the Repository

```sh
git clone https://github.com/EremosCore/Eremos.git
cd Eremos
```

---

## 3. Install Dependencies

```sh
npm install
```

---

## 4. Run the Demo

```sh
npm run demo
```

---
